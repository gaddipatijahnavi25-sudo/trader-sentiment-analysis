# Trader Performance vs Market Sentiment Analysis

## Overview
This project analyzes how Bitcoin market sentiment (Fear/Greed Index) influences trader behavior and profitability on Hyperliquid.

## Methodology

1. Cleaned and merged sentiment and trading datasets.
2. Aggregated trader metrics at daily level.
3. Compared performance across sentiment regimes.
4. Segmented traders by frequency, size, and consistency.

## Key Metrics Created
- Daily PnL per trader
- Win rate
- Trades per day
- Average trade size
- Long/Short ratio

## Key Insights

1. Trader profitability is highest during Fear periods.
2. Trade frequency increases significantly during Fear.
3. High-frequency traders outperform in volatile markets.

## Strategy Recommendations

### Strategy 1:
During Fear days, increase trade frequency and favor short bias.

### Strategy 2:
During Neutral markets, reduce trading and focus on high conviction setups.
